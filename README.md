# 🎯 Smart Client

A lightweight browser-based performance overlay with:

- FPS tracking
- Ping monitoring
- Lag detection
- Auto optimization system
- Draggable UI
- Live performance graphs

## 🚀 Getting Started

1. Download or clone repo
2. Open `index.html` in browser

## 🧠 Features

- Real-time FPS tracking
- Network latency monitoring
- Performance status detection
- Auto optimization simulation
- Canvas-based graphs

## ⚠️ Notes

This is an experimental client for performance monitoring.
No game injection or bypass systems included.
